Hermes Agent er ikke bare en chatbot - den er en autonom AI-agent som kan:

  • Kjoere terminal-kommandoer pa maskinen din
  • Sende meldinger pa Telegram, Discord, Slack
  • Lese og skrive filer, kode, dokumenter
  • Soek paa nettet og hente informasjon
  • Styre GitHub repos (du har 19 repos med full PAT!)
  • Huske ting mellom sessions (via SOUL.md, MEMORY.md, Helix)
  • Delege oppgaver til sub-agenter (Asha'man)

  • Identitet - Hvem er agenten? (Hermes / MajereAI / OWL)

  • Always-active skills - 35 skills som alltid lastes
  • Bruker-instruksjoner - Kommunikasjonsstil, autonomi

Obsidian - Ditt Eksterne Minne
Laer aa bruke notatsystemet som integreres med Hermes
Hvorfor Obsidian?

Obsidian er et kraftig notatsystem basert paa Markdown-filer. Det fungerer som ditt "eksterne minne" - en kunnskap-base du selv eier og kontrollerer.

Agenten kan lage, soek og redigere Obsidian-notater via obsidian skillen:

# Les en note
python3 -c "
from pathlib import Path
vault = Path.home() / 'Obsidian'
note = vault / 'Dagbok.md'
if note.exists():
    print(note.read_text())
"

# Soek i notater
grep -r "Hermes" ~/Obsidian/ --include="*.md"

# Lag ny note
cat > ~/Obsidian/Hermes-Laeringslogg.md << 'EOF'
# Hermes Laeringslogg

## 2026-05-30
- Installert Hermes Agent pa BMAX
- Konfigurert Telegram bot
- 35 always-aktive skills
EOF

Hva er en LLM?

En Large Language Model (LLM) er et nevraalt neaert trent paa enormous mengder tekst. Den lærer aa forutsi neste ord i en sekvens - og evnen til aa gjoere det gir den "intelligens".

Input:  "Hjulet av tid"
Context: "Hjulet av tid snur, og epoker kommer gaar..."
Output: "...minner som blir legende"

# Parametre = "minne" modellen har laert
# Owl Alpha: 1M tokens kontekstvindu
# Temperatur = kreativitet (0.0 = deterministisk, 1.0 = kreativ)
Viktige Begreper
Prompt

Instruksjonen du gir modellen

Tokens

Ord/bokstaver modellen leser (1 token ~ 4 bokstaver)

Context Window

Hvor mye tekst den kan "huske" (Owl Alpha: 1M tokens)

Temperature

Kreativitet vs presisjon (0.0 - 1.0)

System Prompt

Gjemte instruksjoner (SOUL.md!)

Tools

Funksjoner modellen kan kalle (terminal, web, filer)
